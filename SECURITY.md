# Security policy

This source snapshot has no configured public repository or private security
contact. Before publishing a release, the distributor must add a monitored
private reporting channel here. Until that exists, do not deploy this alpha in a
security-sensitive or regulated production workflow. Never include
patient-identifiable or proprietary analysis data in a report.

PyMixEF evaluates only its documented safe formula grammar. Interchange parsers
preserve or reject code-like constructs; they do not execute NM-TRAN or R code.
The package emits no telemetry.

The 0.1 series is pre-stable. Security fixes are released on the latest 0.1
version; a long-term-support policy will be defined before a stable release.
